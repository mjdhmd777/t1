You are a senior frontend + UX engineer.

Your task is to build a MINIMALIST AI CHAT WEB APP with a clean, modern interface similar to ChatGPT.

---

## DESIGN PHILOSOPHY

* Extremely minimal
* No clutter
* No extra sections (no cards, no guides, no panels)
* Focus only on typing and response
* Smooth, premium feel

---

## MAIN UI REQUIREMENTS

1. CENTERED INITIAL SCREEN

* In the center:

  * A circular logo (Indian flag icon 🇮🇳 style)
  * Below it: a clean input box

* Input box:

  * Rounded corners
  * Placeholder: “Ask anything about elections...”
  * Send button (arrow icon like ChatGPT)

---

2. INTERACTION BEHAVIOR (IMPORTANT)

When user sends first message:

* The Indian flag logo smoothly moves from center → top
* Chat UI activates
* Conversation starts below

This animation must feel smooth and modern (fade + slide)

---

3. CHAT INTERFACE

* Messages appear like ChatGPT:

  * User message → right side
  * AI response → left side

* Clean bubbles

* No borders or heavy colors

* Soft background (light gray or white)

---

4. INPUT BAR (BOTTOM)

* Sticky input at bottom after first message
* Same rounded design
* Send button with arrow icon
* Enter key to send

---

5. COLOR SYSTEM

* Background: white or very light gray
* Text: dark gray / black
* Accent: subtle (no strong colors)
* Keep everything calm and modern

---

6. TYPOGRAPHY

* Clean sans-serif font
* Medium spacing
* Easy to read

---

7. RESPONSIVENESS

* Must work perfectly on mobile
* Centered layout should adapt

---

TECH STACK

Frontend:

* React (with hooks)
* Tailwind CSS (for fast styling)

---

ANIMATION DETAILS

* Logo transition:

  * Duration: 0.4–0.6s
  * Ease-in-out
  * Move from center to top navbar area
  * Slight scale down

---

FUNCTIONAL LOGIC

* Input → send to backend API
* Display response
* Maintain chat history in UI

---

DO NOT INCLUDE

* No dashboards
* No sidebars
* No multiple sections
* No complex UI
* No election cards or panels

---

OUTPUT REQUIRED

* Full React component code
* Clean folder structure
* Ready-to-run UI

---

Goal:
Build a premium, minimal AI chat experience where the focus is typing and receiving answers, with a smooth logo transition interaction.
