You are a senior frontend engineer and UX designer.

Your task is to upgrade the existing AI chat web app into a visually appealing, smooth, and modern experience — while keeping it minimalist.

---

## DESIGN GOAL

Make the app feel:

* Smooth
* Modern
* Premium
* Interactive
* Lightweight (not cluttered)

Do NOT add heavy UI or unnecessary sections.

---

## CORE PRINCIPLES

* Minimal design first
* Subtle animations (not flashy)
* Focus on user input and conversation
* Every animation must have a purpose

---

## ANIMATION ENHANCEMENTS

1. LOGO ANIMATION (IMPORTANT)

* Indian flag logo in center on load
* When user sends first message:

  * Smoothly move logo to top
  * Slight scale down
  * Add fade transition
  * Duration: 0.5s ease-in-out

---

2. MESSAGE ANIMATIONS

* User message:

  * Slide in from right
  * Fade in

* AI message:

  * Fade + slight upward motion

* Add typing effect for AI response (streaming feel)

---

3. INPUT BOX INTERACTION

* Soft glow or border highlight on focus
* Smooth hover effect on send button
* Button press animation (scale down slightly)

---

4. SUGGESTION BUTTONS (CHIPS)

* Appear with fade + stagger animation

* Hover effect:

  * Slight scale up
  * Background subtle change

* On click:

  * Smooth ripple or press animation

---

5. LOADING STATE (IMPORTANT)

When AI is thinking:

* Show subtle animated dots or shimmer
* Example:
  "Thinking..." with animated dots (...)

Make it feel alive, not static

---

6. BACKGROUND ENHANCEMENT

* Add very subtle gradient (white → light gray or light blue)
* Optional:
  Soft floating blur shapes (very low opacity)

---

7. SCROLL BEHAVIOR

* Smooth auto-scroll to latest message
* No jumpy movement

---

8. MICRO-INTERACTIONS

* Button hover transitions
* Smooth transitions between states
* Slight delay for realism

---

TECH IMPLEMENTATION

* Use Framer Motion for animations
* Use Tailwind CSS for styling
* Keep performance optimized

---

DO NOT ADD

* No dashboards
* No complex layouts
* No heavy illustrations
* No too many colors

---

OUTPUT REQUIRED

* Updated React components with animations
* Clean and reusable code
* Smooth UX improvements

---

GOAL

Transform the app from basic to premium by using motion, spacing, and subtle interactions — without breaking simplicity.
